Spustite program install.bat priamo z diskety.
Problemy pri instalacii :
Q : Aj ked som spustil install, ikony sa mi vo windows neobjavili. Co mam robit?
A : - Windows 3.1 .. Staci kliknut na Subor(File)/Novy(New)/Jednotlivy program(Application) a nasledne na tlacitko Prochazet(Browse). Nastavte sa na adresar z matematikou a na subor matemat.exe. Kliknite na tlacitko Ok, este raz Ok a je to...
- Windows 95 .. Pravym tlacitkom kliknite kdekolvek na prac. plochu a nasledne zvolte z menu Novy objekt(New)/Zastupce(ShortCut). Tlacitkom Prochazet(Browse) nalistujte na Vasom disku program, OK, Ok a je to.
Q : Ked som obdrzal reg. cislo, ako sa zaregistrujem?
A : Jednoducho : Spustite program Matematika a z menu zvolte Extra/Nastavenia a na zalozke vyberte volbu Registracia. Do prveho policka napiste svoje meno a do druheho registracne cislo. Nasledne musite kliknut na tlacitko Registracia (hned pod reg. cislom).
Q : Nejdu mi obrazky z geometrie. Co s tym?
A : Zvolte z menu Extra/Nastaveni a kliknite na zalozku Adresare. Do policka Obrazky napiste spravny adresar k obrazkom *.wmf (su sucastov instalacie) a kliknite na Ulozit nastavenia a Ok.
Q : Vo Windows 3.11 sa mi v niektorych dialogoch objavuju vacsie pismena ako rozmer okienka...
A : Znamena to, ze mate problemy s fontami. Asi mate zapnute Large fonts. Riesenim je bud prepnut Windows do modu min. 800x600 so slovenskymi small fontami, alebo pridat prislusne fonty priamo do Ovladacieho Panelu (Control Panel) v zlozke Pisma (Fonts).
Q : Nejde mi slovenska diakritika!!!
A : Doinstalujte si fonty s koncovkou CE do Vasej verzie Windows, alebo si priamo nainstalujte ceske Windows. Ak nejde diakritika v helpe, nainstalujte font cetimes.ttf (alebo vo Windows 95 CE staci font times.ttf).
